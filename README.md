# 🎓 Teaching Assistance Performance Prediction

## 📌 Project Overview
This project focuses on analyzing Teaching Assistant (TA) evaluation data and building Machine Learning models to predict the performance category of teaching assistants.

The performance categories are:
- Low
- Medium
- High

The project includes complete Exploratory Data Analysis (EDA), data preprocessing, model building, model evaluation, and comparison of multiple Machine Learning algorithms.

---

## 🎯 Problem Statement

### Task 1
Prepare a complete data analysis report on the given dataset.

### Task 2
Build a predictive Machine Learning model to accurately predict the performance of students/Teaching Assistants.

## 📂 Dataset Description

The dataset contains evaluations of Teaching Assistant performance over:
- Three regular semesters
- Two summer semesters

Total Records:
- 151 Teaching Assistant assignments

### Features Used

| Feature | Description |
|---|---|
| Native_teacher | Whether TA is a native English speaker |
| Instructor | Course instructor category |
| Course | Course category |
| Semester | Summer or Regular semester |
| Class_size | Number of students in class |
| Class_attribute | Target variable (Low/Medium/High) |
